# 💊 PharmacyApp — Online Pharmacy Ordering System

A modern **JavaFX desktop application** that simulates the customer experience of an online pharmacy. Users can browse medicines, manage a shopping cart, place pickup or delivery orders, and track their complete order history through an intuitive and visually polished interface.

The application was designed using a **mobile-first user experience approach**, inspired by contemporary e-commerce and medicine delivery platforms, while being implemented as a desktop prototype using Java and JavaFX.

## ✨ Key Features

* 🔐 User authentication with sign up and login validation
* 💊 Browse and search available medicines
* 🛒 Shopping cart with quantity management and voucher support
* 📦 Checkout with Delivery and Pickup fulfillment options
* 🚚 Delivery scheduling with address management
* 💳 Multiple payment methods
* 📋 Complete order history with filtering and search
* ❌ Order cancellation and deletion functionality
* 💾 Persistent storage across application restarts

## 🏗️ Architecture

The project follows Object-Oriented Design principles including encapsulation, composition, abstraction, and separation of concerns. Model classes manage application data, while the JavaFX application layer handles presentation and user interactions.
